# 단어장 (Wordbook)

![Vibe Coded](https://img.shields.io/badge/vibe%20coded-with%20Claude-8b6f47)

> 이 프로젝트는 Claude와 함께한 바이브 코딩(vibe coding)으로 작성되었습니다.

순우리말·유의어를 정리하고 단어 사이의 관계를 탐색하는 단일 파일 HTML 단어장입니다. 별도의 서버나 빌드 과정 없이 브라우저에서 바로 동작하며, GitHub Gist를 백엔드 삼아 여러 기기 간 데이터를 동기화합니다.

## 특징

- **서버리스 단일 파일** — `index.html` 하나로 모든 기능(UI·로직·스타일)이 동작합니다. 정적 호스팅이나 로컬 파일로 바로 열 수 있습니다.
- **GitHub Gist 동기화** — 개인 액세스 토큰(PAT)으로 비공개 Gist에 단어를 저장하고, Gist ID를 공유해 다른 기기에서도 같은 단어장을 불러옵니다. 데이터는 본인 GitHub 계정에만 보관됩니다.
- **초성 분리 저장** — 단어를 한글 초성(ㄱ~ㅎ) 그룹으로 나눠 `vocab_ㄱ.json`, `vocab_ㄴ.json` … 형태의 여러 파일로 저장합니다. 단어가 많아져도 변경된 그룹만 다루도록 설계되어 있습니다.
- **유의어 백링크** — 단어 A의 유의어로 B를 등록하면, B를 볼 때 "A에서 참조됨"이 자동으로 표시됩니다. 단어 간 양방향 관계를 따라가며 탐색할 수 있습니다.
- **하버카드 미리보기** — 유의어 칩에 마우스를 올리면 해당 단어의 유의어를 모달에서 나가지 않고 미리 볼 수 있습니다. 등록된 단어와 참조만 된 단어를 점(●/○)으로 구분합니다.
- **검색·초성 탭·페이지네이션** — 단어/유의어 검색, 초성별 필터링, 페이지 분할을 지원합니다.
- **자동 단어 생성(선택)** — 유의어를 입력하면 그 유의어들도 자동으로 단어로 등록하고 서로를 유의어로 연결하는 옵션을 제공합니다.
- **로컬 백업** — 전체 데이터를 JSON 파일로 내보내고 다시 가져올 수 있습니다. `localStorage`에도 항상 캐시되어 오프라인 조회가 가능합니다.

## 시작하기

1. `index.html`을 브라우저에서 엽니다. (GitHub Pages 등 정적 호스팅에 올려도 됩니다.)
2. GitHub 동기화를 쓰려면 `gist` 권한을 가진 [개인 액세스 토큰](https://github.com/settings/tokens/new?description=Vocabulary%20Sync&scopes=gist)을 생성해 설정 화면에 입력합니다.
3. 처음 저장하면 비공개 Gist가 자동 생성됩니다. 발급된 **Gist ID**를 다른 기기 설정에 입력하면 같은 단어장을 공유합니다.

자세한 설정 과정(GitHub 가입, 토큰 발급, 다중 기기 연결, 백업/복원, FAQ)은 [`manual.html`](manual.html)에 단계별 안내가 정리되어 있습니다.

## 데이터 구조

단어는 다음과 같은 형태로 저장됩니다.

```json
{
  "연민": { "synonyms": ["불쌍히 여기다", "측은지심"] },
  "측은지심": { "synonyms": ["연민"] }
}
```

Gist에는 초성 그룹별 `vocab_*.json` 파일과 메타 정보용 `vocab_meta.json`이 함께 보관됩니다.

## 기술 스택

- 순수 HTML / CSS / JavaScript (프레임워크·빌드 도구 없음)
- GitHub Gist REST API
- 브라우저 `localStorage`

## 보안 메모

GitHub 토큰은 외부로 전송되지 않고 브라우저 `localStorage`에만 저장되어 GitHub API 호출에만 사용됩니다. 공용 PC에서는 사용 후 토큰을 삭제하는 것을 권장합니다.
