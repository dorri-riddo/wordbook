GitHub 동기화 설정

🔒 비공개 저장소
데이터는 Secret Gist로 저장되어 URL을 아는 사람만 접근 가능합니다.

여기를 클릭해서 토큰 생성 → 'gist' 권한만 체크 → 생성

다른 기기에서 동일한 데이터를 보려면 이 ID를 입력하세요

다른 기기에서 생성한 Gist ID가 있다면 여기에 입력하세요

체크하면 입력한 동의어가 자동으로 새 낱말로 추가됩니다

처음 설정하는 경우

  1. 위 링크로 GitHub 토큰 생성 페이지 이동
  2. 'gist' 권한만 체크하고 토큰 생성
  3. 생성된 토큰을 복사해서 위에 입력
  4. '저장' 버튼 클릭

다른 기기에서 동일한 데이터 사용하기

  1. 첫 번째 기기에서 생성된 Gist ID를 '복사' 버튼으로 복사
  2. 두 번째 기기에서 동일한 토큰 입력
  3. '기존 Gist ID'에 복사한 ID 붙여넣기
  4. '저장' 클릭 → 자동으로 데이터 동기화

새 낱말 추가

쉼표(,)로 구분해서 입력하세요
